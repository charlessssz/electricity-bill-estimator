# Electricity Bill Estimator (C++)

## Description
A C++ console-based Electricity Bill Estimator that calculates electricity consumption using manual input or appliance-based computation. It supports full CRUD operations (add, view, update, delete) for appliances.

## Features
- Manual electricity bill calculation
- Appliance-based energy computation
- Add appliances
- View appliances
- Update appliance details
- Delete appliances
- kWh computation

## Formula Used
kWh = (Power × Hours × Days) / 1000

## How to Run
1. Compile using g++:
   g++ main.cpp -o app

2. Run the program:
   ./app
